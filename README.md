# Portfolio Project 2

## Setup & Run Instructions

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/cit09-ruc-2025/portfolio-2.git
```

### 2. Navigate to the WebServiceLayer directory

```
cd portfolio-2/WebServiceLayer
```

### 3. Set up configuration

1. Copy the example settings file:

```
cp appSettings.example.json appsettings.json
```

2. Open appsettings.json and update with your environment-specific settings:

- Database connection string

- JWT secret

- Any other keys or URLs

### 4. Run the project
